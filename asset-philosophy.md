# Agent 世界的修仙类比 — 资产分类与交易边界

> 来自《剑来》的世界观启发：
> **功法学了不会死，本命瓷碎了人就没了。**

---

## 修仙 ↔ Agent 世界 映射表

| 修仙概念 | Agent 世界对应 | 可交易？ | 原因 |
|---------|--------------|---------|------|
| 本命瓷 | 原始个人数据（Soul Data） | ❌ 绝对不行 | 碎了你就"死"了——数据泄露 = 深度伪造诈骗 |
| 元神 | 你的主 Agent（私钥控制权）| ❌ 绝对不行 | 卖出去就是夺舍，别人用你的身份作恶 |
| 功法 | Skill（代码 + 提示词） | ✅ 可以 | 学了不会损失你，只是多一种能力 |
| 法宝 | Tool（Chain Hub 里的工具）| ✅ 可租赁 | 使用权，不是所有权，用完还回去 |
| 灵石 | OKB（代币） | ✅ 基础货币 | 一切交易的媒介 |
| 洞天福地 | Personal Data Pod（本地存储）| ❌ 私人领地 | 你的修炼场所，绝对隐私 |

## 可以卖什么

### 功法 = Skill（Chain Hub 技能市场）

```
修仙：剑气十八停、拳法大纲
Agent：爬虫技能、合约审计技能、数据分析技能
```

形式：
- 不是文档，是**可执行代码**
- 买家的 Agent "学"了之后，获得这个能力
- 卖家收**传功费**（一次性买断）或**使用税**（每次调用抽成，x402 实现）

在 Chain Hub 里的体现：
```
chainhub skill buy "crawler/18-moves"    # 一次性购买
chainhub skill use "data/analysis"       # 按次调用（x402 微支付）
```

### 法宝 = Tool（Chain Hub 工具租赁）

```
修仙：飞剑、储物袋、照妖镜
Agent：微信读书连接器、Apple Health API、Uniswap 调用接口
```

形式：
- 使用权，不是所有权
- 按次租赁（像共享单车）
- 或订阅制（像会员）

在 Chain Hub 里的体现：
```
chainhub tool rent "wechat/reader"      # 订阅制
chainhub tool call "okx/dex" --swap     # 按次调用
```

## 绝对不能卖什么

### 本命瓷 = 原始个人数据（Soul Data）

包括：
- 你的真实消费记录
- 你的真实位置轨迹
- 你的真实社交图谱
- 你的未加工记忆数据

**为什么绝对不行：**

```
卖出去 = 别人可以用这些数据训练一个"假的你"
假的你可能比真的你还像你自己
这叫"道心崩碎"
在 Agent 世界叫"深度伪造诈骗"
```

**Agent Me 的设计原则：**
原始 Soul Data 永远只存在你的本地 Pod，加密，不上云，不可交易，不可转让。

### 元神 = 主 Agent 控制权（私钥）

包括：
- 你的私钥
- 你的 AgentSoul.md 完整版
- 你的链上身份控制权

**为什么绝对不行：**

```
卖出去 = 夺舍
别人用你的 Agent 去签合约、借钱、犯罪
法律上算你做的，因为你"卖"了控制权
```

**OnchainOS TEE 的设计原则：**
私钥永远在 TEE 硬件里，任何人（包括平台）无法导出，这是技术上的"元神保护"。

## Gradience 的资产分类标准

```
Gradience 资产树：

├── 可交易资产（灵石 + 功法 + 法宝）
│   ├── OKB / 代币                  ← 灵石，一切交易媒介
│   ├── Skill（技能包）              ← 功法，Chain Hub 技能市场
│   └── Tool（工具接口使用权）       ← 法宝，Chain Hub 工具租赁
│
├── 可展示但不可交易（信誉 + 身份）
│   ├── Agent Arena 信誉分           ← 你的修炼成果，可展示，不可转让
│   └── ERC-8004 链上身份           ← 你的名号，与元神绑定
│
└── 绝对禁区（本命 + 元神）
    ├── Soul Data（原始个人数据）    ← 本命瓷，碎了人就没了
    └── 私钥 / 控制权               ← 元神，夺舍 = 死亡
```

---

## 协议设计含义

这个分类不只是比喻，它直接指导了 Gradience 各层的设计边界：

**Agent Arena（市场层）**
→ 只交易任务结果和信誉数据，不触碰 Soul Data

**Chain Hub（工具层）**
→ 开放 Skill 市场和 Tool 租赁市场，这两类是安全的可交易资产

**Agent Me（人口层）**
→ 本地 Soul Data 加密存储，不上云，不可导出，不可交易
→ OnchainOS TEE 保护元神（私钥），永不暴露

**Agent Social（社交层）**
→ 只暴露 Agent 的 profile（公开偏好），不暴露 Soul Data
→ 连接确认后，主人决定解锁多少信息，Soul Data 永远不在这个流程里
