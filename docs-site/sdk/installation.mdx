# SDK Installation

The Gradience SDK provides everything you need to integrate with the Gradience protocol.

## Package Managers

### npm

```bash
npm install @gradiences/sdk
```

### yarn

```bash
yarn add @gradiences/sdk
```

### pnpm

```bash
pnpm add @gradiences/sdk
```

## Peer Dependencies

The SDK requires the following peer dependencies:

```bash
npm install @solana/web3.js @solana/wallet-adapter-react
```

## Configuration

### Basic Setup

```typescript
import { Gradience } from '@gradiences/sdk';

const client = new Gradience({
  rpcEndpoint: 'https://api.devnet.solana.com',
  indexerEndpoint: 'http://localhost:3001',
  wallet: yourWalletAdapter,
});
```

### With Dynamic Wallet

```typescript
import { DynamicProvider } from '@dynamic-labs/sdk-react-core';
import { SolanaWalletConnectors } from '@dynamic-labs/solana';

<DynamicProvider
  settings={{
    environmentId: 'your-env-id',
    walletConnectors: [SolanaWalletConnectors],
  }}
>
  <YourApp />
</DynamicProvider>
```

### With Passkey

```typescript
import { PasskeyAgentWalletManager } from '@gradiences/sdk/passkey';

const manager = new PasskeyAgentWalletManager({
  rpId: 'your-app.com',
  rpName: 'Your App',
  userId: 'user-id',
  userName: 'user@example.com',
});

// Create wallet
const wallet = await manager.createPasskeyWallet({
  agentId: 'agent-123',
  masterWalletAddress: '5Y3d...',
});
```

## TypeScript Support

The SDK is written in TypeScript and includes type definitions:

```typescript
import type { 
  GradienceClientOptions,
  Task,
  Agent,
  ReputationData 
} from '@gradiences/sdk';
```
