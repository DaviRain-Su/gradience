# GRA-119: [ChainHub] Deploy program to Solana devnet

## Description
Build BPF binary and deploy Chain Hub program to Solana devnet. Record program ID in project config.

## Steps
1. Install Solana CLI + Anchor (if needed)
2. `cargo build-bpf` in `apps/chain-hub/program`
3. Create devnet keypair for program
4. `solana program deploy` to devnet
5. Record program ID in `apps/chain-hub/sdk/` and `packages/chain-hub-sdk/`
6. Verify with `solana program show <program-id>`

## Acceptance Criteria
- [ ] BPF build succeeds
- [ ] Program deployed to devnet
- [ ] Program ID recorded in SDK config
- [ ] Smoke test: initialize instruction callable on devnet
