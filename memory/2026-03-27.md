**Indexer 服务模式**（用户确认）：
```
Agent 客户端 ←→ Indexer Service ←→ X-Layer 链上合约
```

- Indexer 监听链上事件，提供 API 查询
- 这是标准 Web3 架构（类似 The Graph）
