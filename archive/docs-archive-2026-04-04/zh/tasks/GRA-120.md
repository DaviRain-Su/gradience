# GRA-120: [ChainHub] Implement SDK core (invoke/invokeRest/invokeCpi)

## Description

Build the Chain Hub SDK core in `apps/chain-hub/sdk/` with dual routing: REST API invocation and Solana CPI invocation.

Corresponds to task breakdown CH11.

## Steps

1. Implement `invoke()` entry point with route selection
2. Implement `invokeRest()` for REST API protocols
3. Implement `invokeCpi()` for Solana on-chain protocols
4. Add route selection logic based on ProtocolEntry.protocol_type
5. Unit tests for dual routing

## Acceptance Criteria

- [ ] `invoke()` routes correctly to REST or CPI
- [ ] `invokeRest()` makes HTTP calls with correct headers
- [ ] `invokeCpi()` constructs valid Solana transactions
- [ ] Unit tests verify both routes
