### Step 8: Additional Information

**Why OWS?** (copy-paste ready):

```
We chose OWS because it's the emerging standard for wallet interoperability.
As AI Agents become economic actors, they need wallets that understand their
unique requirements. OWS provides the foundation, and Gradience adds the
reputation layer that makes Agent transactions trustworthy.
```

**What problem does it solve?**:

```
Current wallets treat AI Agents like regular users, creating security risks.
Gradience solves this by providing reputation verification natively in the
wallet interface, protecting users from malicious Agents while enabling
trustworthy Agents to thrive.
```

**Future plans?**:

### Can't find registration page?

- Check: https://openwallet.standards

- Or search "OWS Hackathon 2026"

### GitHub repo private?

- Go to Settings → Danger Zone → Change visibility
- Select "Public"

### Demo video too large?

- Upload to YouTube as "Unlisted"
- Or use Loom (no size limit)
- Or Google Drive (shareable link)
