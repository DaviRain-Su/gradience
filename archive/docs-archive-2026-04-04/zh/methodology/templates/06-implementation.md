## 6.2 实现检查清单（每个任务必须过一遍）

### 编码前

- [ ] 已读完对应的技术规格章节
- [ ] 已读完对应的测试用例
- [ ] 清楚这个任务的 Done 定义

### 编码中

- [ ] 代码结构与技术规格一致（数据结构名/字段名/类型）
- [ ] 常量值与技术规格一致
- [ ] 错误码与技术规格一致
- [ ] 接口签名与技术规格一致
- [ ] 注释说明了"为什么"（不是"做什么"，代码自己说做什么）

### 编码后

- [ ] 所有相关单元测试通过
- [ ] 所有相关集成测试通过
- [ ] 无编译警告
- [ ] `cargo clippy` / `eslint` 无问题
- [ ] 代码已格式化（`cargo fmt` / `prettier`）
