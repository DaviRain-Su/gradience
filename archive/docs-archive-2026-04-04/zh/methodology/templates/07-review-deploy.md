# Phase 7: Review & Deploy（审查与部署）

> **目的**: 确保代码质量和安全性，部署到目标环境
> **输入**: Phase 6 的代码 + 测试结果
> **输出物**: 审查报告 + 部署记录，存放到 `<project>/docs/07-review-report.md`

---

## 7.1 代码审查（必填）

### 7.1.1 自审清单

- [ ] 代码与技术规格 100% 一致
- [ ] 所有公开接口有文档注释
- [ ] 无硬编码的密钥/密码/敏感信息
- [ ] 无未处理的 TODO/FIXME（如有，已记录为后续任务）
- [ ] 错误处理完整（无 unwrap/panic/未捕获异常）
- [ ] 日志/事件适当（可观测性）

### 7.1.2 安全审查

| 检查项         | 状态  | 备注 |
| -------------- | ----- | ---- |
| 权限校验完整   | ✅/❌ |      |
| 无重入风险     | ✅/❌ |      |
| 整数运算安全   | ✅/❌ |      |
| PDA 种子无碰撞 | ✅/❌ |      |
| 账户关闭后清零 | ✅/❌ |      |
| 外部调用安全   | ✅/❌ |      |
