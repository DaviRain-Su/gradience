## Visa Requirements

### US Visa Requirements by Nationality

| Nationality           | Visa Required? | Type    | Processing Time | Notes              |
| --------------------- | -------------- | ------- | --------------- | ------------------ |
| US Citizens           | No             | N/A     | N/A             | Domestic travel    |
| Visa Waiver Countries | ESTA           | Online  | 72 hours        | Valid 2 years      |
| Other Countries       | B1/B2          | Embassy | 2-8 weeks       | Interview required |

### Visa Waiver Program (VWP) Countries

Citizens of the following countries can apply for ESTA instead of a visa:

- Most EU countries, UK, Australia, Japan, South Korea, Singapore, etc.
- Full list: https://travel.state.gov/content/travel/en/us-visas/tourism-visit/visa-waiver-program.html

### ESTA Application Checklist (for VWP nationals)

- [ ] Apply online: https://esta.cbp.dhs.gov/
- [ ] Have valid passport ready
- [ ] Pay fee ($21 USD)
- [ ] Save approval confirmation
- [ ] Apply at least 72 hours before travel

### B1/B2 Visa Checklist (if required)

- [ ] Complete DS-160 form online
- [ ] Pay visa fee ($185 USD)
- [ ] Schedule embassy interview
- [ ] Prepare documents:
    - [ ] Valid passport
    - [ ] DS-160 confirmation
    - [ ] Visa appointment letter
    - [ ] Photo (2x2 inches)
    - [ ] Hackathon invitation letter
    - [ ] Employment verification
    - [ ] Bank statements
    - [ ] Return ticket proof
- [ ] Attend interview
- [ ] Collect passport with visa
