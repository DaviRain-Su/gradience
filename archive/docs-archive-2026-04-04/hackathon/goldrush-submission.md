**Project Name**: Gradience - Reputation-Powered Agent Wallet

**Tagline**: The First Social Wallet with Native Agent Identity & Risk Scoring

**Description**:

```
Gradience integrates GoldRush's blockchain data API to build a
reputation-powered wallet for AI Agents. Our solution uses on-chain
data to calculate trust scores, verify Agent credibility, and enable
secure Agent-to-Agent transactions.

Key innovation: Real-time risk scoring using GoldRush data +
Chain Hub reputation oracle + OWS wallet standard.
```
