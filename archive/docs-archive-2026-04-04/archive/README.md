## 归档规则

文档在满足以下条件时会被归档：

1. **时间过期** - Hackathon 计划等活动已结束
2. **决策完成** - 比较分析等决策文档已产生结论
3. **工具更替** - 工具或流程已被新方案取代（如 Linear → Obsidian）
4. **内容整合** - 内容已被更新、更完整的文档取代
5. **问题已解决** - 经验报告中的问题已修复
