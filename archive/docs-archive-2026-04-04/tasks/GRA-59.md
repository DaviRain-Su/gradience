# GRA-59: [OWS] Build reputation-powered wallet MVP

## Description

构建声誉驱动的钱包 MVP

## 核心功能

1. Wallet 连接
    - 支持主流钱包（MetaMask, Phantom, etc.）
    - OWS 标准接口
2. Agent 声誉展示
    - 连接 Gradience Chain Hub
    - 实时查询声誉评分
    - 可视化展示（分数/等级）
3. 交易保护
    - 交易前声誉检查
    - 低声誉警告提示
    - 白名单/黑名单管理
4. Agent 发现
    - 浏览高声誉 Agents
    - 搜索和筛选

## 技术实现

- Frontend: Next.js + Tailwind
- Backend: Gradience SDK
- Blockchain: Solana/Ethereum multi-chain
