# GRA-204: 实现 Ephemeral Rollups (ER) 核心集成

## Description

实现 MagicBlock Ephemeral Rollups 的核心功能：创建 ER 会话、在 ER 中执行操作、提交回 L1。

### 白皮书定义

> MagicBlock Ephemeral Rollups extend Solana's settlement capacity:
> - 0 gas within ER
> - <10ms finality
> - 10,000+ TPS
> - State auto-commits back to Solana L1

## Implementation

### 1. 重构 MagicBlockExecutionEngine

```typescript
import { EphemeralRollup, ERSession } from '@magicblock-labs/ephemeral-rollups-sdk';

export class MagicBlockExecutionEngine {
    private er: EphemeralRollup;
    private sessions = new Map<string, ERSession>();

    constructor(config: MagicBlockConfig) {
        this.er = new EphemeralRollup({
            endpoint: config.erEndpoint,
            apiKey: config.apiKey,
        });
    }

    async createSession(accounts: PublicKey[]): Promise<ERSession> {
        const session = await this.er.createSession({
            accounts: accounts.map(a => a.toBase58()),
            ttlSeconds: 3600,  // 1 hour
        });
        this.sessions.set(session.id, session);
        return session;
    }

    async executeInER(
        sessionId: string,
        instructions: TransactionInstruction[]
    ): Promise<string> {
        const session = this.sessions.get(sessionId);
        if (!session) throw new Error('Session not found');

        const result = await session.execute(instructions);
        return result.signature;
    }

    async commitToL1(sessionId: string): Promise<string> {
        const session = this.sessions.get(sessionId);
        if (!session) throw new Error('Session not found');

        const result = await session.commit();
        return result.l1Signature;
    }
}
```

### 2. 高频结算场景

```typescript
// 批量 judgeAndPay 在 ER 中执行
async function batchJudgeInER(
    engine: MagicBlockExecutionEngine,
    tasks: JudgeTaskInput[]
): Promise<BatchJudgeResult> {
    // 1. 收集所有相关账户
    const accounts = tasks.flatMap(t => [t.taskPda, t.escrowPda, t.winnerPda]);
    
    // 2. 创建 ER 会话
    const session = await engine.createSession(accounts);
    
    // 3. 构建所有 judgeAndPay 指令
    const instructions = await Promise.all(
        tasks.map(t => buildJudgeAndPayIx(t))
    );
    
    // 4. 在 ER 中批量执行
    for (const ix of instructions) {
        await engine.executeInER(session.id, [ix]);
    }
    
    // 5. 提交回 L1
    const l1Sig = await engine.commitToL1(session.id);
    
    return { sessionId: session.id, l1Signature: l1Sig, count: tasks.length };
}
```
