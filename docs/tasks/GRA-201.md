# GRA-201: 添加 Payment Channels UI（A2A 微支付）

## Description

为 A2A Protocol 的微支付通道添加前端 UI。允许 Agent 之间建立支付通道，进行高频小额支付。

### 功能点

1. **通道管理**
    - 创建通道 (openChannel)
    - 查看活跃通道
    - 关闭通道 (cooperativeClose)

2. **链下签名状态**
    - 显示当前 nonce
    - 显示已花费/剩余
    - 双方签名状态

3. **争议处理**
    - 发起争议 (openDispute)
    - 争议状态追踪
    - 仲裁结果

## Implementation

1. 创建 `PaymentChannelsView.tsx`
2. 创建 `ChannelCard.tsx`
3. 添加 hooks:
    - `useOpenChannel`
    - `useCloseChannel`
    - `useChannelState`
4. 集成 A2A Protocol SDK

## A2A SDK 调用

```typescript
// 打开通道
const openIx = await sdk.openChannel({
    channelId: BigInt(Date.now()),
    depositAmount: 0.5 * LAMPORTS_PER_SOL,
    expiresAt: deadline,
    payee: counterpartyAddress,
});

// 协作关闭
const closeIx = await sdk.cooperativeCloseChannel({
    channelId,
    nonce: latestNonce,
    spentAmount,
    payerSig,
    payeeSig,
});
```
