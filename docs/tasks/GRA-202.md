# GRA-202: MagicBlock Ephemeral Rollups 集成

## Description

集成 MagicBlock 的 Ephemeral Rollups (ER) 作为结算层的可选增强，用于高频结算场景。

### 白皮书定义

> MagicBlock Ephemeral Rollups extend Solana's settlement capacity without changing the protocol's trust model.

| 能力 | 说明 |
|------|------|
| ER | 弹性结算加速：0 gas, <10ms finality, 10,000+ TPS |
| PER | Private ER via Intel TDX TEE，隐私评判 |
| VRF | 链上可验证随机数，公平 Judge 选择 |

### 使用场景

- 高频批量结算（Arena 锦标赛）
- 隐私评判（sealed submissions）
- 随机 Judge 轮换

## Implementation

1. 添加 MagicBlock SDK
   ```bash
   pnpm add @magicblock/sdk
   ```

2. 创建 `magicblock-adapter.ts`
   - ER 会话管理
   - State delegation
   - Commit back to L1

3. 修改结算流程
   - 检测是否启用 ER
   - 高频场景自动切换

## MagicBlock API

```typescript
import { createEphemeralSession, delegateState } from '@magicblock/sdk';

// 创建 ER 会话
const session = await createEphemeralSession({
    programId: GRADIENCE_PROGRAM_ADDRESS,
    accounts: [taskPda, escrowPda],
});

// 在 ER 中执行
await session.execute(judgeAndPayIx);

// 提交回 L1
await session.commit();
```
