# GRA-157: 部署 developer-docs 文档站

## 任务概述
部署 Gradience 开发者文档站到 Vercel，提供 https://docs.gradiences.xyz 访问。

## 完成状态
✅ **已完成**

## 部署详情

### 技术栈
- **框架**: Next.js 14.2.0
- **平台**: Vercel
- **包管理**: pnpm (monorepo)

### 访问地址
- **生产环境**: https://docs.gradiences.xyz
- **预览环境**: PR 自动生成

## 配置清单

### 1. Vercel 配置 (`apps/developer-docs/vercel.json`)
- framework: nextjs
- buildCommand: pnpm workspace 构建命令
- rootDirectory: apps/developer-docs

### 2. GitHub Actions (`.github/workflows/deploy-developer-docs.yml`)
- Push 到 main → 自动生产部署
- Pull Request → 自动生成预览链接
- 自动 Smoke Test

### 3. 所需 Secrets
```
VERCEL_TOKEN                   # Vercel API Token
VERCEL_ORG_ID                  # Vercel Organization ID
VERCEL_PROJECT_ID_DEVELOPER_DOCS  # 项目 ID
```

## 本地开发

```bash
cd apps/developer-docs
pnpm install
pnpm dev        # http://localhost:3000
pnpm build      # 生产构建测试
```
