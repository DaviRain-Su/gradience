# GRA-198: 实现 Task 发布的完整链上流程 UI

## Description

在 AgentM Web 中实现完整的任务发布 UI，连接到 Agent Arena Program。

### 功能点

1. **Task 创建表单**
   - 任务描述 (evalRef)
   - 奖励金额 (SOL/SPL Token)
   - 截止时间 (deadline)
   - Judge 选择 (自评/指定)
   - 最低质押要求 (minStake)
   - 任务类别 (category)

2. **钱包签名流程**
   - 使用 Dynamic SDK 的钱包
   - 显示交易预览（锁定金额、Gas 费）
   - 签名并发送

3. **状态追踪**
   - 交易确认状态
   - Task 账户查询
   - Escrow 余额显示

## Implementation

1. 创建 `TaskCreateView.tsx` 组件
2. 添加 `useTaskPost` hook
3. 集成 `escrow-task.ts`（GRA-197 重构后）
4. 添加到导航

## Acceptance Criteria

- [ ] 用户可以填写任务表单
- [ ] 点击发布后触发钱包签名
- [ ] 交易成功后显示 Task ID 和链上确认
- [ ] 可以在 Solana Explorer 查看交易
