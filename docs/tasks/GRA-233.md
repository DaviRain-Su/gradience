# GRA-233: Bundle 优化 — 移除未使用依赖

## 背景

当前 bundle 大小 789KB，目标 <500KB。主要问题：
- `@privy-io/react-auth` 引入但未使用 (+200KB)
- 其他未使用依赖

## 目标

将 bundle 大小从 789KB 减少到 <500KB。

## 优化方案

### 1. 移除未使用依赖
```bash
# 检查未使用依赖
npx depcheck

# 确认移除
- @privy-io/react-auth
- 其他未使用包
```

### 2. 代码分割
```typescript
// 动态导入大组件
const HeavyComponent = dynamic(() => import('./HeavyComponent'), {
  loading: () => <Skeleton />
});
```

### 3.  Tree Shaking 优化
- 检查 sideEffects 配置
- 优化导入方式

### 4. 图片优化
- 使用 Next.js Image
- WebP 格式
- 响应式图片

## 验收标准

- [ ] Bundle 分析报告
- [ ] 移除 @privy-io/react-auth

- [ ] Lighthouse 性能评分 > 90
