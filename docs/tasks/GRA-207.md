# GRA-207: 实现 VRF 可验证随机 Judge 选择

## Description

实现 MagicBlock VRF（可验证随机函数）用于公平、不可预测的 Judge 选择。

### 白皮书定义

> VRF (Verifiable Random): On-chain verifiable randomness for Judge selection. Ensures fair, unpredictable Judge assignment in competitive tasks.

## Implementation

### 1. VRF 集成

```typescript
import { VRFClient, VRFProof } from '@magicblock-labs/vrf-sdk';

export class VRFJudgeSelector {
    private vrf: VRFClient;

    constructor(config: MagicBlockConfig) {
        this.vrf = new VRFClient({
            endpoint: config.erEndpoint,
            apiKey: config.apiKey,
        });
    }

    async selectJudge(
        taskId: string,
        candidates: string[],
        seed?: Uint8Array
    ): Promise<VRFSelectionResult> {
        // 1. 请求 VRF 随机数
        const vrfResult = await this.vrf.requestRandomness({
            seed: seed || this.generateSeed(taskId),
            numWords: 1,
        });

        // 2. 使用随机数选择 Judge
        const randomIndex = Number(vrfResult.randomWords[0] % BigInt(candidates.length));
        const selectedJudge = candidates[randomIndex];

        // 3. 返回结果和证明
        return {
            judge: selectedJudge,
            proof: vrfResult.proof,
            randomness: vrfResult.randomWords[0],
            verifiable: true,
        };
    }

    private generateSeed(taskId: string): Uint8Array {
        // 使用 taskId + blockhash 作为 seed
        const encoder = new TextEncoder();
        return encoder.encode(`${taskId}:${Date.now()}`);
    }
}

interface VRFSelectionResult {
    judge: string;
    proof: VRFProof;
    randomness: bigint;
    verifiable: boolean;
}
```

### 3. 链上验证合约

```rust
// Solana Program 中的验证
pub fn verify_judge_selection(
    ctx: Context<VerifyJudge>,
    vrf_proof: [u8; 64],
    randomness: u64,
) -> Result<()> {
    // 验证 VRF proof
    let valid = verify_vrf(
        &ctx.accounts.vrf_pubkey.key(),
        &vrf_proof,
        &randomness.to_le_bytes(),
    )?;
    
    require!(valid, ErrorCode::InvalidVRFProof);
    
    // 确认 judge 选择正确
    let candidates = &ctx.accounts.task.judge_candidates;
    let expected_index = (randomness % candidates.len() as u64) as usize;
    require!(
        ctx.accounts.judge.key() == candidates[expected_index],
        ErrorCode::WrongJudgeSelected
    );
    
    Ok(())
}
```

### 4. Judge 轮换机制

```typescript
interface JudgeRotationPolicy {
    rotationInterval: number;  // 每 N 个任务轮换
    minCandidates: number;     // 最少候选人数
    excludeRecent: number;     // 排除最近 N 次的 Judge
}

export class JudgeRotationManager {
    private history: string[] = [];

    async selectNextJudge(
        candidates: string[],
        policy: JudgeRotationPolicy,
        vrfSelector: VRFJudgeSelector
    ): Promise<VRFSelectionResult> {
        // 排除最近的 Judge
        const eligible = candidates.filter(
            c => !this.history.slice(-policy.excludeRecent).includes(c)
        );

        if (eligible.length < policy.minCandidates) {
            throw new Error('Not enough eligible judges');
        }

        const result = await vrfSelector.selectJudge('rotation', eligible);
        this.history.push(result.judge);

        return result;
    }
}
```
