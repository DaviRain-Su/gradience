# GRA-173: [AgentM Web] Agent 智能配置界面 (P0)

## 描述
在 Agent 创建页面添加智能配置模式，用户可以用自然语言描述需求，AI 自动生成配置界面

## 背景
这是 P0 核心功能。当前 Agent 配置使用固定表单，用户需要了解所有字段。智能配置允许用户用自然语言描述，AI 生成针对性界面。

## 使用场景
```
用户输入: "我想监控 ETH 价格，当低于 2000 美元时通知我"
         ↓
AI 生成: TokenSelector(ETH) + PriceInput(2000) + NotificationType
         ↓
用户: 调整参数 → 提交
```

## 技术细节

### 组件接口
```typescript
interface SmartConfigProps {
  onSubmit: (config: AgentConfig) => void;
  initialMode?: "traditional" | "smart";
}
```

### 状态管理
```typescript
type SmartConfigState = 
  | "idle"           // 等待输入
  | "generating"     // AI 生成中
  | "preview"        // 显示生成的表单
  | "editing"        // 用户编辑中
  | "submitting";    // 提交中
```
