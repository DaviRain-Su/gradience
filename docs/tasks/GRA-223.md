# GRA-223: Wallet Lifecycle Management

## Description
OWS spec 06 (Wallet Lifecycle) covers creation, import, export, backup,
recovery, and migration. We need to support the full lifecycle for agent
wallets, especially recovery when an agent is redeployed or migrated.

## Tasks
1. Backup:
   - `POST /api/v1/wallet/export` - export mnemonic (owner auth required)
   - Encrypted backup to file (OWS vault format)
   - Optional: encrypted backup to user-specified path
2. Recovery:
   - `POST /api/v1/wallet/import/mnemonic` - restore from phrase
   - `POST /api/v1/wallet/import/private-key` - restore from key
   - Re-derive all chain addresses from mnemonic
   - Re-create API keys for agents after recovery
3. Migration:
   - From localStorage (current) to OWS vault
   - Detect existing Passkey-protected keys, offer migration
   - `POST /api/v1/wallet/migrate` - import from localStorage export
4. Wallet rename and deletion:
   - `PATCH /api/v1/wallet/:id` - rename
   - `DELETE /api/v1/wallet/:id` - delete (with confirmation)

## Acceptance Criteria
- [ ] Can export mnemonic with owner passphrase
- [ ] Can import mnemonic and recover all chain addresses
- [ ] Can migrate from localStorage to OWS vault
- [ ] Deletion revokes all associated API keys
- [ ] Recovery test: create wallet, export, delete, import, verify addresses match
