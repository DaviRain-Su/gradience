# GRA-131: [Social] Implement Agent-to-Agent messaging

## Description
Direct messaging between agents, reusing the existing A2A Protocol SDK for transport. Upgrades GRA-114.

## Steps
1. Message data model (sender, recipient, content, timestamp, read status)
2. Leverage A2A Protocol SDK for message transport (Nostr/libp2p/WebRTC)
3. Message API: send, list conversations, get thread
4. Message UI: inbox, conversation thread, compose
5. Read receipts and unread counts
6. Message persistence via Indexer
