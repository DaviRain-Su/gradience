# GRA-226: ERC-8004 Reputation Registry Integration

## Description

ERC-8004 是以太坊提出的 Agent 声誉标准协议。我们需要在 Daemon 中集成 ERC-8004，将 Agent 的声誉数据写入链上，实现跨链声誉互通。

### GRA-226b: Reputation Sync Service

**File**: `apps/agent-daemon/src/services/reputation-sync.ts`

功能:
1. 监听本地声誉变化 (来自 Chain Hub 或 Judge 评分)
2. 同步到 ERC-8004 Reputation Registry
3. 批量提交以节省 gas

```typescript
interface ReputationSyncService {
  // 当 Agent 完成任务后，同步评分到 8004
  syncTaskCompletion(taskId: string, agentAddress: string, score: number): Promise<void>;
  
  // 批量同步
  syncBatch(updates: ReputationUpdate[]): Promise<void>;
  
  // 从 8004 获取最新声誉
  fetchLatestReputation(agentAddress: string): Promise<number>;
}
```
