# GRA-147: [Code] MagicBlock 从 A2ARouter 移出，重构为 ExecutionEngine

## Description
将 MagicBlock 从 A2ARouter 的通信适配器重构为结算层的可选增强模块。不再是独立的"执行层"，而是 Solana 结算操作的加速/隐私组件。

## 变更
- 从 A2ARouter adapters 中移除 MagicBlockAdapter
- 创建 apps/agentm/src/main/settlement/magicblock-enhancement.ts
- 实现生命周期: Delegate → Execute in ER → Commit → Undelegate
- 区分 ER (普通执行) 和 PER (sealed/TEE 模式)
- 提供 VRF 接口用于 Judge 随机选择
- 关联到 ChainHub 合约调用

## 文件
- apps/agentm/src/main/a2a-router/adapters/magicblock-adapter.ts (移除)
- apps/agentm/src/main/settlement/ (新建)

## Acceptance Criteria
- [ ] MagicBlock 不再是 A2ARouter adapter
- [ ] ExecutionEngine 实现 Delegate→Execute→Commit 生命周期
- [ ] ER 和 PER 模式可切换
- [ ] 编译通过
