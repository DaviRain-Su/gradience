# GRA-171: [AgentM Web] json-render 基础架构搭建

## 描述
安装 json-render 依赖，配置基础架构，创建 Catalog 和 Registry

## 背景
这是 json-render 集成的第一阶段，需要建立完整的技术基础，包括依赖安装、组件库定义和注册表配置。

## 子任务
- [ ] 安装依赖包
  - @json-render/core
  - @json-render/react
  - @json-render/shadcn
  - @json-render/next
- [ ] 创建 components/json-render/catalog.ts
  - 定义所有可用组件
  - 配置组件 props schema
  - 添加组件描述
- [ ] 创建 components/json-render/registry.tsx
  - 绑定实际组件实现
  - 配置 action handlers
- [ ] 创建 components/json-render/types.ts
  - 定义类型扩展
  - 导出常用类型
- [ ] 定义 AgentM 组件库
  - 基础组件: Card, Button, Input, Select, Slider
  - 自定义组件: TokenSelector, PriceChart, MetricCard
  - 布局组件: Grid, Stack, Flex

## 验收标准
- [ ] 所有依赖安装成功，无冲突
- [ ] Catalog 定义完整，包含所有计划组件
- [ ] TypeScript 类型检查通过
- [ ] 基础渲染测试通过
- [ ] 代码通过 lint 检查

## 技术细节

### 文件结构
```
components/json-render/
├── catalog.ts          # 组件库定义
├── registry.tsx        # 组件注册表
├── types.ts            # 类型定义
└── utils.ts            # 工具函数
```

### 关键代码
```typescript
// catalog.ts 示例
export const agentmCatalog = defineCatalog(schema, {
  components: {
    Card: { props: z.object({ title: z.string() }) },
    TokenSelector: { props: z.object({ chainId: z.number() }) },
  },
});
```
