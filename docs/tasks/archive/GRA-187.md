# GRA-187: Extract shared A2A types to packages/a2a-types

## Description

Create `packages/a2a-types/` as a shared type-only package that both `agent-daemon` and `agentm` (Electron) can import. Currently `a2a-router-types.ts` and `nostr-types.ts` live in `apps/agentm/src/shared/` -- they need to become a proper workspace package.

## Files to Create

- `packages/a2a-types/package.json`
- `packages/a2a-types/tsconfig.json`
- `packages/a2a-types/src/index.ts` (re-exports)
- `packages/a2a-types/src/a2a-router-types.ts` (move from agentm/src/shared/)
- `packages/a2a-types/src/nostr-types.ts` (move from agentm/src/shared/)
- `packages/a2a-types/src/constants.ts` (move NOSTR_CONFIG, ROUTER_CONFIG, error codes)

## Acceptance Criteria

- [ ] Package compiles with `tsc`
- [ ] No runtime dependencies (types + constants only)
- [ ] `apps/agentm/` imports from `@gradiences/a2a-types` instead of local shared/
- [ ] `apps/agent-daemon/` can import from `@gradiences/a2a-types`
