# GRA-172: [AgentM Web] AI Spec 生成服务

## 描述

实现后端 API 服务，将自然语言转换为 json-render Spec

## 背景

这是整个 json-render 集成的核心服务层。用户用自然语言描述需求，AI 需要生成对应的 JSON Spec。

## 技术细节

### API 接口

```typescript
POST /api/ai/generate-spec
Body: {
  prompt: string;
  context: {
    type: "agent-config" | "dashboard" | "playground";
    userId?: string;
  };
}
Response: {
  spec: JsonSpec;
  cached?: boolean;
}
```

### Prompt 示例

```typescript
const AGENT_CONFIG_PROMPT = `You are an expert UI designer...

User Description: {{userDescription}}
Available Components: {{componentCatalog}}

Generate a valid JSON Spec...`;
```
