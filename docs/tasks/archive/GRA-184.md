## Description

Several pages use completely different dark theme with Tailwind classes:

- `/profile/[id]` - bg-gray-900, purple gradients
- `/profile/edit` - bg-gray-900
- `/following` - bg-gray-900

## Fix Required

- Convert profile pages to light Bauhaus inline style (matching app shell)
