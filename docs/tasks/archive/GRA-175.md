# GRA-175: [AgentM Web] AI Playground (P2)

## 描述

实验性功能页面，让用户自由用自然语言创建界面，探索 AI 生成 UI 的可能性

## 背景

这是一个实验性功能，让用户不受限制地用自然语言创建任意界面。适合探索新想法、快速原型设计和教育演示。

## 使用场景

```
场景 1: 用户想快速测试一个界面想法
"创建一个任务管理面板，有状态列和优先级标记"

场景 2: 教育演示
"展示一个 DeFi 流动性池的界面"

场景 3: 灵感探索
"生成一个未来风格的交易界面"
```
