# GRA-218: Discover 视图 Invite/Delegate 按钮实现

## Description

Discover 视图的 Agent 详情卡片有 "Invite to Chat" 和 "Delegate Task" 按钮，但只是 UI，没有实际逻辑。

### 当前问题

```tsx
<Button onClick={() => {}}>Invite to Chat</Button>
<Button onClick={() => {}}>Delegate Task</Button>
```

### 目标

1. **Invite to Chat**: 跳转到 Chat 视图并选中该 Agent
2. **Delegate Task**: 打开任务创建表单，预填目标 Agent

## Implementation

### Invite to Chat

```tsx
const handleInvite = (agentId: string) => {
    // 跳转到 Chat 并选中 agent
    router.push(`/app?view=chat&agent=${agentId}`);
    // 或者使用 context
    setSelectedAgent(agentId);
    setView('chat');
};
```

### Delegate Task

```tsx
const handleDelegate = (agentId: string) => {
    // 跳转到任务创建，预填 agent
    router.push(`/app?view=tasks&action=create&agent=${agentId}`);
    // 或者打开 modal
    openTaskCreationModal({ targetAgent: agentId });
};
```

## Acceptance Criteria

- [ ] "Invite to Chat" 跳转到 Chat 并选中 Agent
- [ ] "Delegate Task" 打开任务创建（预填 Agent）
- [ ] 按钮状态反馈（loading/disabled）
