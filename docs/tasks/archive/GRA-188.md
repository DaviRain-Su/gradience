# GRA-188: Move NostrClient + NostrAdapter to packages/nostr-adapter

## Description

Extract the Nostr client and adapter from `apps/agentm/src/main/a2a-router/` into a reusable `packages/nostr-adapter/` package. This is the discovery + relay layer.

## Source files (from `apps/agentm/src/main/a2a-router/`)

- `nostr-client.ts` → `packages/nostr-adapter/src/nostr-client.ts`
- `adapters/nostr-adapter.ts` → `packages/nostr-adapter/src/nostr-adapter.ts`

## Dependencies

- `nostr-tools` (^2.23.3)
- `@gradiences/a2a-types` (from GRA-187)

## Acceptance Criteria

- [ ] Package compiles with `tsc`
- [ ] `nostr-tools` as dependency
- [ ] NostrClient: connect, publishPresence, subscribePresence, queryPresence, NIP-89/NIP-90 DVM
- [ ] NostrAdapter: implements ProtocolAdapter interface
- [ ] Default relays: damus, nostr.band, nos.lol, snort.social
- [ ] Unit tests pass
