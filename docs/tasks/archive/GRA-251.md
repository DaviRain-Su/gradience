# GRA-251: [EVM] Configure EVM networks in Foundry

## Description

Configure Foundry (`foundry.toml`) to support multi-chain EVM deployments. Primary target is XLayer Testnet (Chain ID 195) and Mainnet (Chain ID 196), with additional RPC profiles for Base, Arbitrum, and Ethereum to ensure the deployment pipeline is chain-agnostic.

## Acceptance Criteria

- [ ] `foundry.toml` includes `[rpc_endpoints]` for `xlayer`, `xlayer-testnet`, `base`, `arbitrum`, `ethereum`
- [ ] `ETHERSCAN_API_KEY` or equivalent verifier config is set for contract verification
- [ ] `PRIVATE_KEY` is loaded via `.env` or cast wallet
- [ ] `forge script` can deploy to any configured network
- [ ] Anvil fork tests run against XLayer testnet RPC
