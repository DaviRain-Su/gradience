# GRA-209: 实现 LLM-as-Judge 真正的评估逻辑

## Description

当前 `evaluator/judges.ts` 中的所有 Judge 都是 **Mock 实现**，使用 `Math.random()` 生成分数。需要实现真正的评估逻辑。

### 当前问题

## Implementation

### 1. CodeJudge - 真正的代码评估

```typescript
protected async performEvaluation(task: EvaluationTask): Promise<EvaluationScores> {
    const repoPath = await this.cloneRepo(task.submission.source);

    // 1. 编译检查
    const buildResult = await this.runCommand('npm run build', repoPath);

    // 2. 运行测试
    const testResult = await this.runCommand('npm test -- --coverage', repoPath);
    const coverage = this.parseCoverage(testResult.stdout);

    // 3. Lint 检查
    const lintResult = await this.runCommand('npm run lint', repoPath);
    const lintScore = this.calculateLintScore(lintResult);

    // 4. 安全扫描
    const securityResult = await this.runCommand('npm audit --json', repoPath);
    const securityScore = this.calculateSecurityScore(securityResult);

    return { buildResult, testResult, coverage, lintScore, securityScore };
}
```

### 2. ContentJudge - LLM 评估

```typescript
protected async performEvaluation(task: EvaluationTask): Promise<EvaluationScores> {
    const content = await this.fetchContent(task.submission.source);
    const criteria = task.evaluationCriteria;

    const prompt = `
You are a content quality evaluator. Score the following content on these criteria:
- Accuracy (0-100): Is the information correct?
- Clarity (0-100): Is it easy to understand?
- Completeness (0-100): Does it cover all required topics?
- Originality (0-100): Is it unique and valuable?

Content to evaluate:
${content}

Evaluation criteria from task:
${criteria}

Respond in JSON format:
{ "accuracy": N, "clarity": N, "completeness": N, "originality": N, "feedback": "..." }
`;

    const response = await this.llmClient.chat({
        model: 'gpt-4',
        messages: [{ role: 'user', content: prompt }],
    });

    return JSON.parse(response.content);
}
```

### 3. UIJudge - Playwright 真正集成

```typescript
// 当前已有 PlaywrightHarness，但需要真正的视觉对比
protected async performEvaluation(task: EvaluationTask): Promise<EvaluationScores> {
    const result = await this.playwright.verifyUI({
        url: task.submission.source,
        expectedScreenshot: task.submission.metadata?.referenceScreenshot,
        accessibilityCheck: true,
        responsiveBreakpoints: [375, 768, 1024, 1440],
    });

    // 使用 LLM 进行视觉评估
    if (task.submission.metadata?.referenceDesign) {
        const visualScore = await this.llmVisualCompare(
            result.screenshots,
            task.submission.metadata.referenceDesign
        );
        result.scores.push({ name: 'Visual Match', score: visualScore });
    }

    return result;
}
```

### 4. LLM Client 抽象

```typescript
// evaluator/llm-client.ts
export interface LLMClient {
    chat(request: ChatRequest): Promise<ChatResponse>;
    embed(text: string): Promise<number[]>;
}

export class OpenAIClient implements LLMClient {
    constructor(private apiKey: string) {}

    async chat(request: ChatRequest): Promise<ChatResponse> {
        const response = await fetch('https://api.openai.com/v1/chat/completions', {
            method: 'POST',
            headers: {
                Authorization: `Bearer ${this.apiKey}`,
                'Content-Type': 'application/json',
            },
            body: JSON.stringify({
                model: request.model || 'gpt-4',
                messages: request.messages,
            }),
        });
        return response.json();
    }
}
```
