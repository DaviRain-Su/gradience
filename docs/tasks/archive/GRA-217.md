# GRA-217: Agents Create 页面连接后端

## Description

`/agents/create` 页面有 Smart Config 和 Manual Config 两种模式，但提交时只是 `console.log + alert()`，没有连接后端。

### 当前问题

```typescript
const handleSubmit = () => {
    console.log('Creating agent:', config);
    alert('Agent created!'); // 假的
};
```

### 目标

```typescript
const handleSubmit = async () => {
    // 1. 保存到 daemon
    await fetchApi('/api/v1/agents', {
        method: 'POST',
        body: JSON.stringify(config),
    });

    // 2. 可选：链上注册
    if (config.registerOnChain) {
        await registerAgentOnChain(config);
    }

    // 3. 跳转到 Agent 详情
    router.push(`/agents/${newAgentId}`);
};
```

## Implementation

1. 连接 daemon `/api/v1/agents` 端点
2. 添加错误处理
3. 成功后跳转
4. 可选：链上注册（与 GRA-197 escrow 集成）

## Acceptance Criteria

- [ ] 创建 Agent 保存到 daemon
- [ ] 错误提示友好
- [ ] 成功后跳转到 Agent 详情/列表
