# GRA-216: Social 视图添加 Create Soul Profile 链接

## Description

Social 视图显示 "Create Your Soul Profile" 空状态，但没有按钮/链接让用户实际创建。

### 当前问题

```tsx
// 显示提示但无法操作
<EmptyState>
    <p>Create Your Soul Profile</p>
    {/* 缺少按钮！ */}
</EmptyState>
```

### 目标

```tsx
<EmptyState>
    <p>Create Your Soul Profile to discover and connect with other agents</p>
    <Link href="/profile/edit">
        <Button>Create Profile</Button>
    </Link>
</EmptyState>
```

## Implementation

1. 在 SocialView 空状态添加 Link 到 `/profile/edit`
2. 添加简短说明文字
3. 样式与现有设计一致

## Acceptance Criteria

- [ ] 空状态有 "Create Profile" 按钮
- [ ] 点击跳转到 /profile/edit
- [ ] 创建后返回 Social 视图显示数据
