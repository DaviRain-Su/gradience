# GRA-214: Task Market 移除 MOCK_TASKS fallback

## Description

Task Market 视图在 daemon 离线时显示假数据（8 个硬编码任务），应该改为显示空状态（像 Discover 一样）。

### 当前问题

```typescript
// 当 daemon 离线时，fallback 到假数据
const MOCK_TASKS = [
    { id: 'task-1', title: 'Build DeFi Dashboard', reward: 500, ... },
    // ... 8 个假任务
];

// 应该改为空状态
if (!tasks.length) {
    return <EmptyState message="No tasks available" />;
}
```

## Implementation

1. 找到 `MOCK_TASKS` 定义位置
2. 移除 fallback 逻辑
3. 显示空状态 UI（参考 DiscoverView）
4. "Post Task" 按钮保留，引导用户创建任务

## Acceptance Criteria

- [ ] daemon 离线时显示空状态，不是假数据
- [ ] 空状态有 "Post Task" 引导
- [ ] 移除 MOCK_TASKS 常量
