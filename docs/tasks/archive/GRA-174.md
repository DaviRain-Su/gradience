# GRA-174: [AgentM Web] 动态数据仪表盘 (P1)

## 描述
实现用户查询驱动的动态数据展示界面，用户用自然语言查询，AI 自动选择最佳图表类型

## 背景
传统仪表盘使用固定布局。动态仪表盘允许用户用自然语言查询数据，AI 根据查询内容自动选择合适的图表和布局。

## 使用场景
```
用户输入: "显示我过去 7 天的交易量和收益"
         ↓
AI 分析: 需要交易量图表 + 收益图表 + 关键指标卡片
         ↓
生成: BarChart(交易量) + LineChart(收益) + MetricCards
```

## 技术细节

### 数据流
```
User Query
    ↓
NLP Parser (optional)
    ↓
AI Spec Generator
    ↓
Data Fetcher (indexer API)
    ↓
json-render Renderer
    ↓
React Components
```

### 支持的查询类型
```typescript
type QueryIntent =
  | "profit_analysis"      // 收益分析
  | "volume_analysis"      // 交易量分析
  | "price_tracking"       // 价格追踪
  | "portfolio_overview"   // 持仓概览
  | "transaction_history"  // 交易历史
  | "risk_metrics";        // 风险指标
```
