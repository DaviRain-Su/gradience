# GRA-192: E2E test -- two daemons communicate via Nostr + XMTP

## Description

Create an E2E test that proves two agent-daemon instances on different ports can:
1. Both connect to Nostr relays and discover each other
2. Exchange a direct message via XMTP
3. Route a task_proposal message end-to-end

## Test Plan

```
1. Start Daemon A on port 7420
2. Start Daemon B on port 7421
3. Daemon A broadcasts capabilities via Nostr
4. Daemon B discovers Daemon A via GET /api/v1/a2a/agents
5. Daemon B sends direct_message to Daemon A via POST /api/v1/a2a/send
6. Verify Daemon A received the message in GET /api/v1/a2a/messages
7. Daemon A sends task_proposal to Daemon B
8. Verify Daemon B received it and it appears in task queue
```

## Files to Create

- `apps/agent-daemon/tests/e2e/a2a-two-daemons.test.ts`

## Acceptance Criteria

- [ ] Two daemon instances start on different ports
- [ ] Agent discovery via Nostr works between them
- [ ] Direct message delivery via XMTP works
- [ ] Task messages route to task queue
- [ ] Test completes in < 60 seconds
