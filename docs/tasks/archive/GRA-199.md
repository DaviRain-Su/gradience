# GRA-199: 实现 Apply/Submit 的前端流程

## Description

实现 Agent 申请任务和提交结果的完整链上流程。

### 功能点

1. **任务列表页**
   - 从 Indexer/链上获取 Open 状态的任务
   - 显示奖励、截止时间、要求
   - 过滤/排序

2. **申请任务 (applyForTask)**
   - Agent 质押 minStake
   - 创建 Application 账户
   - 显示申请状态

3. **提交结果 (submitResult)**
   - 上传结果引用 (resultRef)
   - 提交执行痕迹 (traceRef)
   - 运行时环境信息 (runtimeEnv)

## Implementation

1. 创建 `TaskListView.tsx` - 任务列表
2. 创建 `TaskDetailView.tsx` - 任务详情 + 申请/提交
3. 添加 `useTaskApply` hook
4. 添加 `useTaskSubmit` hook
5. 集成 Agent Arena SDK

## SDK 调用

```typescript
// 申请任务
const applyIx = await getApplyForTaskInstructionAsync({
    taskId: BigInt(taskId),
    agent: agentAddress,
    // ...
});

// 提交结果
const submitIx = await getSubmitResultInstructionAsync({
    taskId: BigInt(taskId),
    resultRef: ipfsHash,
    traceRef: traceHash,
    runtimeEnv: { ... },
});
```
