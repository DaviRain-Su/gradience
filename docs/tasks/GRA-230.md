# GRA-230: 多 Agent 任务协调器 UI

## 背景

MultiAgentTaskView 之前调用不存在的 `/api/v1/coordinator/` 端点（GRA-152 已移除）。现在需要重新设计并实现多 Agent 任务协调功能。

## 目标

创建完整的多 Agent 任务协调器界面，支持：

1. 创建多 Agent 任务（分解子任务）
2. 分配子任务给不同 Agent
3. 监控整体进度
4. 协调 Agent 间通信

## 功能规格

### 1. 创建多 Agent 任务

- 任务描述输入
- 自动/手动子任务分解
- Agent 选择（基于技能匹配）
- 预算分配

### 2. 任务看板

- Kanban 视图：待办/进行中/已完成
- 甘特图：时间线视图
- 列表视图：详细状态

### 3. Agent 协调

- 子任务分配
- 进度同步
- 冲突解决
- 结果聚合

### 4. 通信中心

- Agent 间消息
- 人工介入
- 决策记录

## 技术方案

```
页面: /app/coordinator/
组件:
  - TaskCreator: 创建多 Agent 任务
  - TaskBoard: Kanban/甘特图视图
  - AgentSelector: 智能 Agent 推荐
  - ProgressTracker: 实时进度
  - MessageHub: 协调消息中心

API:
  POST /api/v1/coordinator/tasks
  GET  /api/v1/coordinator/tasks/:id
  POST /api/v1/coordinator/tasks/:id/assign
  WS   /ws/coordinator/:taskId
```
