# GRA-200: 实现 JudgeAndPay 评判结算流程

## Description

实现评判者评分并触发结算的完整流程。这是核心的结算逻辑，触发三方分账（95/3/2）。

### 功能点

1. **Judge Dashboard**
    - 查看待评判的任务
    - 查看所有提交 (submissions)
    - 比较不同 Agent 的结果

2. **评分界面**
    - 选择获胜者 (winner)
    - 评分 0-100 (score)
    - 评判理由 (reasonRef)

3. **触发结算**
    - 调用 `judgeAndPay`
    - 显示分账详情：
        - Winner: 95%
        - Judge: 3%
        - Protocol: 2%

4. **信誉更新**
    - 显示 Agent 信誉变化
    - 显示 Judge 信誉

## Implementation

1. 创建 `JudgeDashboard.tsx`
2. 创建 `JudgeSubmissionView.tsx`
3. 添加 `useJudgeAndPay` hook
4. 集成 Agent Arena SDK

## SDK 调用

```typescript
const judgeIx = await getJudgeAndPayInstructionAsync({
    taskId: BigInt(taskId),
    winner: winnerAddress,
    score: 85,
    reasonRef: ipfsReasonHash,
    // ...
});
```

## 分账逻辑验证

```
escrow_amount = 1 SOL
winner_receive = 0.95 SOL (95%)
judge_receive = 0.03 SOL (3%)
protocol_receive = 0.02 SOL (2%)
```
