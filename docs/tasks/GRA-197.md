# GRA-197: 重构 escrow-task.ts 连接 Agent Arena SDK

## Description

当前 `apps/agentm-web/src/lib/solana/escrow-task.ts` 是一个 demo stub，只做了自己转给自己的模拟操作。需要替换为真正调用 Agent Arena Program 的逻辑。

### 当前问题

```typescript
// 当前是假的
SystemProgram.transfer({
    fromPubkey: payer,
    toPubkey: payer,  // Self-transfer for demo
    lamports: params.rewardLamports,
}),
```

### 目标

使用 `@gradiences/agent-arena-sdk` 的 `getPostTaskInstructionAsync` 真正发布任务并锁定资金到 Escrow PDA。

## Implementation

1. 引入 Agent Arena SDK
   ```typescript
   import { getPostTaskInstructionAsync, GRADIENCE_PROGRAM_ADDRESS } from '@gradiences/agent-arena-sdk';
   ```

2. 重构 `buildTaskEscrowTx` 函数
   - 计算 Task PDA、Escrow PDA
   - 调用 `getPostTaskInstructionAsync`
   - 支持 SOL 和 SPL Token

3. 添加 PDA 推导工具函数
   - `deriveTaskPda(taskId)`
   - `deriveEscrowPda(taskId)`

## Acceptance Criteria

- [ ] 任务发布时 SOL 真正锁定到 Escrow PDA
- [ ] 链上可以查询到 Task 账户状态
- [ ] 支持 devnet 和 mainnet 切换
- [ ] 单元测试覆盖

## Dependencies

- Agent Arena SDK: `apps/agent-arena/clients/typescript/`
- Program deployed on devnet
